# Custom Components & Enhancements

This documentation covers all custom markdown components and styling enhancements available in Raksara. These components allow you to create rich, interactive content with minimal effort.

## Available Components

### 1. **Sortable Tables**
GitHub-flavored markdown tables with built-in column sorting. Click table headers to sort ascending, descending, or reset.

**[Learn more →](/doc/table)**

### 2. **Panels** 
Highlighted content blocks with icons and styling. Perfect for notes, warnings, tips, and important information.

**[Learn more →](/doc/panel)**

### 3. **Containers**
Glassmorphism-styled flex containers with scrolling support. Ideal for organizing content in responsive layouts.

**[Learn more →](/doc/container)**

### 4. **Chips**
Compact, pill-shaped badges with optional icons and labels. Great for tags, status indicators, and shorthand information.

**[Learn more →](/doc/chip)**

---

## Quick Start

All components are rendered during the markdown parsing phase. Use the HTML syntax shown in each component's documentation.

<panel type="info">**Pro tip:** Combine multiple components to create complex, visually rich content blocks.</panel>
