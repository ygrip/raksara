# MitmProxy Grid

**MitmProxy Grid** is a distributed proxy infrastructure for intercepting, recording, and analyzing HTTP/HTTPS traffic across multiple services.

## Features

- Distributed proxy nodes managed via a central dashboard
- Real-time traffic inspection and filtering
- Request/response modification rules
- HAR export for offline analysis

## Tech Stack

- Python 3.11
- mitmproxy
- Docker Compose

## Use Cases

- API debugging across microservices
- Performance analysis and bottleneck identification
- Intercept and modify network traffic through rules
