# Raksara

**Raksara** is a GitHub-native static content platform that transforms any repository into a fully functional blog, portfolio, and personal website.

## Features

- Markdown-powered blog with frontmatter metadata
- Portfolio showcase with project cards
- Full-text client-side search
- Tag and category filtering
- Automatic metadata generation
- GitHub Pages deployment via Actions
- Zero runtime dependencies — pure static files

## Tech Stack

- Vanilla JavaScript
- Marked.js (Markdown rendering)
- Highlight.js (syntax highlighting)
- MiniSearch (client-side search)
- Node.js (build scripts)
- GitHub Actions (CI/CD)

## Philosophy

Write in Markdown. Push to GitHub. Your site is live. No databases, no servers, no complex build tools. Just content and code.
